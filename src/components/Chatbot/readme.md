# BrewBuddy Café Chatbot

This is a small business chatbot built for BrewBuddy Café.

## Features
- Answers questions about opening hours, menu, and location
- Friendly café-style responses
- Safe handling of prices
- Modern chat interface

## Setup Instructions

1. Install Python 3.10+
2. Create a virtual environment
3. Install dependencies:
   pip install -r requirements.txt

4. Create a `.env` file:
   OPENAI_API_KEY=your_openai_key

5. Run the app:
   python app.py

6. Open browser:
   http://127.0.0.1:5000
